# Documentación en Markdown (réplicas para Figma)

Esta carpeta contiene **réplicas en .md** de la documentación que en el repositorio está en **AsciiDoc** (`docs/modules/ROOT/pages/*.adoc`). Figma no lee .adoc; aquí se ofrece el mismo contenido en Markdown para que la IA de Figma o cualquier herramienta que solo acepte .md pueda acceder a wireframes, arquitectura y modelo de negocio.

- **index.md** — Página de inicio del componente (propósito, implementación, enlaces).
- **wireframes.md** — Wireframes en ASCII y guía para Figma (layouts, pantallas).
- **example-business-model.md** — Modelo de negocio de ejemplo (Corporación Coderic, multi-tenant).
- **architecture.md** — Diagramas Mermaid (flujo de registro, schemas, flujo de lectura).
- **nav.md** — Índice de navegación (enlaces a los anteriores).

Las instrucciones completas para construir los prototipos en Figma están en el archivo raíz: **figma.md**.
